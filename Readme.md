Bonjour et merci d'avoir installer Somalia Simulator 2014 sur votre ordinateur !

Si vous êtes sous Windows, vous n'avez qu'à lancer l'exécutable !
bin/Debug/SS14.exe

Si vous êtes sous linux, un Makefile est disponible, vous n'avez qu'a faire :
make clean
make
./bin/jeu

/!\ Ce jeu nécessite les bibliothéques SDL 1.3, SDL_gfx, SDL_ttf, FMODex
